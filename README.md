System Architecture

1. Document ingestion
   - PDF parsing
   - Text cleaning
   - Chunking

2. Hybrid retrieval
   - Dense embeddings
   - BM25 keyword search
   - Reciprocal Rank Fusion

3. Tool-augmented agent
   - Router LLM
   - Tavily web search fallback

4. Conversation memory
   - ConversationSummaryBufferMemory

5. Answer generation
   - Groq LLM
